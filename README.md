# Nuage-mot

### Lexique des effets

```
' ' -> reinit
'a' -> former le mot
'h' -> mot suivant
'z' -> fuir vers le haut
'p' -> créer des nuages
'P' -> défaire le nuages
'o' -> mettre en cercle
'q' -> grossir particules
's' -> reduire particules
```
### Lexique des mouvements

```
mouseX (move) -> rapidité de déplacement
mouseY (move) -> opacité du fond
mouseX (drag) -> translate des particules
```
