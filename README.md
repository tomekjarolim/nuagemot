# Nuage-mot

### Lexique des effets

```
' ' -> reinit
'a' -> former le mot
'h' -> mot suivant
'z' -> fuir vers le haut
'p' -> créer des nuages
'P' -> défaire les nuages
'm' -> créer des lucioles
'M' -> défaire les lucioles
'o' -> mettre en cercle
'q' -> grossir particules -> moche
's' -> reduire particules -> moche
'f' -> créer flocking
'F' -> défaire flocking
'r' -> démarrer la rotation
'R' -> arrêter la rotation
'b' -> démarer bacteries (default)
'B' -> arrêter bacteries
```
### Lexique des mouvements

```
mouseX (move) -> rapidité de déplacement
mouseY (move) -> opacité du fond
mouseX (drag) -> translate des particules
```
